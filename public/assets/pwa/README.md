# PWA Icons Setup Guide

This directory contains PWA (Progressive Web App) icons for Android, iOS, and Windows 11 platforms. These files are excluded from version control to reduce repository size.

## How to Generate Icons

To generate the required PWA icons, follow these steps:

### Option 1: Using PWA Builder (Recommended)

1. Visit [PWA Builder Icon Generator](https://www.pwabuilder.com/imageGenerator)
2. Upload your source icon image (recommended: 1024x1024px PNG with transparent background)
3. Configure the following settings:
    - **Platforms**: Select Android, iOS, and Windows
    - **Generate all sizes**: Enable this option
4. Click "Generate" to create all icon sizes
5. Download the generated icons package
6. Extract the contents and place them in this directory structure:
    ```
    public/assets/pwa/
    ├── android/
    │   ├── android-launchericon-48-48.png
    │   ├── android-launchericon-72-72.png
    │   ├── android-launchericon-96-96.png
    │   ├── android-launchericon-144-144.png
    │   ├── android-launchericon-192-192.png
    │   └── android-launchericon-512-512.png
    ├── ios/
    │   └── [various sizes: 16.png, 20.png, 29.png, ...]
    └── windows11/
        └── [various Windows tile sizes]
    ```

### Option 2: Using RealFaviconGenerator

1. Visit [RealFaviconGenerator](https://realfavicongenerator.net/)
2. Upload your source icon
3. Configure settings for Android, iOS, and Windows
4. Download the generated package
5. Extract and organize files into the directory structure above

### Option 3: Manual Generation

If you prefer manual generation:

1. Start with a high-resolution source image (1024x1024px minimum)
2. Use image editing tools (Photoshop, GIMP, ImageMagick) to resize
3. Generate the following required sizes:
    - **Android**: 48x48, 72x72, 96x96, 144x144, 192x192, 512x512
    - **iOS**: 16, 20, 29, 40, 58, 60, 76, 80, 87, 120, 152, 180, 512, 1024
    - **Windows 11**: Various tile sizes (see `icons.json` for reference)

## Required Directory Structure

```
public/assets/pwa/
├── android/
│   └── [Android launcher icons]
├── ios/
│   └── [iOS app icons]
├── windows11/
│   └── [Windows 11 tile icons]
├── icons.json (reference file - optional)
├── .gitignore
└── README.md
```

## Verification

After generating icons, verify that:

1. All required icon files are present in their respective directories
2. The `manifest.json` file in `/public/manifest.json` references the correct icon paths
3. Icons are properly formatted (PNG format, correct dimensions)

## Notes

-   The `icons.json` file is a reference catalog generated by PWA builder tools and is optional
-   The actual PWA manifest (`/public/manifest.json`) uses these icons
-   Icons should have transparent backgrounds for best results
-   Ensure icons are optimized for web (compressed PNG files)

## Source Icon Requirements

-   **Format**: PNG with transparency
-   **Minimum Size**: 1024x1024 pixels
-   **Recommended**: Square aspect ratio
-   **Design**: Should work well at small sizes (48x48px)
